INTERFACE_HEADER_SETSÂ¶

New in version 3.23.

Read-only list of the target's INTERFACE and PUBLIC header sets (i.e. all file sets with the type HEADERS). Files listed in these header sets can be installed with install(TARGETS) and exported with install(EXPORT) and export().

Header sets may be defined using the target_sources() command FILE_SET option with type HEADERS.

See also HEADER_SETS.
