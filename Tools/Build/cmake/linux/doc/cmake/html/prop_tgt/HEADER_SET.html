HEADER_SETÂ¶

New in version 3.23.

Semicolon-separated list of files in the target's default header set, (i.e. the file set with name and type HEADERS). If any of the paths are relative, they are computed relative to the target's source directory. The property supports generator expressions.

This property is normally only set by target_sources(FILE_SET) rather than being manipulated directly.

See HEADER_SET_<NAME> for the list of files in other header sets.
